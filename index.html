Projeto Integrador de Competências II

Para abrir o Power BI utilize seu email da Universidade Cruzeiro do Sul

exemplo: nome.sobrenome@cs.cruzeirodosul.edu.br

Objetivos:

  1. Obter um banco de dados;
  2. Fazer a ingestão de dados num serviço de nuvem;
  3. Criar um acesso local para manusear esses dados;
  4. Carregar os dados em Power BI e criar um dashboard com os dados.
