Sobre

Projeto Integrador Transdiciplinar II
Apresentação do Power BI

  1. Obter um banco de dados;
  2. Fazer a ingestão de dados em um serviço de nuvem;
  3. Criar um acesso local para manusear esses dados;
  4. Carregar os dados em Power BI e criar um dashboard com os dados.

Dashboard

Para acessar utilize seu e-mail institucional da Universidade Cruzeiro do Sul. nome.sobrenome@cs.cruzeirodosul.edu.br

Informações

O objetivo do trabalho é criar um pipeline de dados, iniciando com o banco de dados, a avaliação do banco, a extração, transformação e carregamento para nuvem, a conexão local para manipular os dados e o painel feito no Power BI.

ERD

Diagrama de relacionamento das entidades, para criar, normalizar e relacionar os dados, utilizando-os no painel ao final.

ETL

Extração e tratamento dos dados para que as tabelas atendam aos requisitos do ERD.

Power BI

Após carregar os dados em nuvem, desenha-se um painel para apresentação amigável das informações.
